# Raw Data Directory

This directory contains raw, immutable datasets used in the projects.

## Guidelines

- **Do not modify** files in this directory
- Keep original data as downloaded
- Document the source of each dataset

## Datasets

Add your datasets here and document them below:

### Example:
- `iris.csv` - Iris flower dataset from UCI ML Repository
- `titanic.csv` - Titanic dataset from Kaggle
- `housing.csv` - Housing prices dataset

## Data Sources

Common sources for datasets:
- [Kaggle Datasets](https://www.kaggle.com/datasets)
- [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/index.php)
- [scikit-learn built-in datasets](https://scikit-learn.org/stable/datasets.html)
- [Google Dataset Search](https://datasetsearch.research.google.com/)

## Note

Large data files (>100MB) should be added to `.gitignore` to avoid bloating the repository.
