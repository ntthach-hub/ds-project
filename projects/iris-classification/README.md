# Iris Classification Project

## Overview
This project uses the classic Iris dataset to demonstrate various classification algorithms and data science techniques.

## Dataset
The Iris dataset contains 150 samples of iris flowers with 4 features:
- Sepal length (cm)
- Sepal width (cm)
- Petal length (cm)
- Petal width (cm)

Target: 3 species (Setosa, Versicolor, Virginica)

## Files
- `iris_classification.py`: Main classification script
- `iris_analysis.ipynb`: Jupyter notebook with detailed analysis and visualization

## Algorithms Implemented
1. Logistic Regression
2. K-Nearest Neighbors (KNN)
3. Decision Tree
4. Random Forest
5. Support Vector Machine (SVM)

## How to Run

### Python Script
```bash
python iris_classification.py
```

### Jupyter Notebook
```bash
jupyter notebook iris_analysis.ipynb
```

## Results
The project compares multiple algorithms and their accuracy scores on the Iris dataset.

- Data loading and preprocessing
- Train-test split
- Model training and evaluation
